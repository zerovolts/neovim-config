# Neovim Cheatsheet

**Leader Key:** `Space`

## General
- `jk` / `kj` — Exit insert mode (Insert)
- `<Esc>` — Clear search highlighting (Normal)
- `<C-d>` — Scroll down and center (Normal)
- `<C-u>` — Scroll up and center (Normal)
- `<C-q>` — Reload vimrc (Normal)

## Window Navigation
- `<Up>` — Window above
- `<Down>` — Window below
- `<Left>` — Window left
- `<Right>` — Window right

## Buffer Navigation
- `<Leader>n` — Next buffer
- `<Leader>p` — Previous buffer

## Terminal
- `<C-j>` — Toggle terminal
- `<Esc>` — Exit terminal mode

## LSP
- `K` — Hover documentation
- `gd` — Go to definition
- `gD` — Go to declaration
- `gi` — Go to implementation
- `go` — Go to type definition
- `gr` — Find references
- `gs` — Show signature help
- `<F2>` — Rename symbol
- `<F3>` — Format document
- `<F4>` — Code actions

## Telescope
- `<Leader>ff` — Find files
- `<Leader>fg` — Live grep
- `<Leader>fb` — Find buffers
- `<Leader>fh` — Find help tags

## Git (Gitsigns)
- `<Leader>hs` — Stage hunk
- `<Leader>hr` — Reset hunk
- `<Leader>hu` — Undo staged changes
- `<Leader>hd` — Diff this file

## Commenting
- `<C-/>` — Toggle line comment (Normal/Visual)

## File Explorer (Oil)
- `<Leader>e` — Open Oil (parent directory of current file)
- `-` — Go up to parent directory (inside Oil)
- `<CR>` — Open file or directory

## UI
- `<Leader>xx` — Toggle Trouble diagnostics

## LÖVE (Love2D)
- `<Leader>v` — LÖVE menu
- `<Leader>vv` — Run LÖVE project
- `<Leader>vs` — Stop LÖVE project
